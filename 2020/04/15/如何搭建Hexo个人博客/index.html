环境配置

安装node.js

node
node.js安装是捆绑npm（包管理器）一起安装的，所以安装完node.js后，npm也安装好了
因为hexo是由node.js支持的
打开终端 win+r 输入cmd
node -v 验证node是否安装成功
npm -v 验证npm

安装hexo是借助npm来安装 但国内安装速度慢 建议安装淘宝镜像cnpm

npm install -g cnpm –registry=https://registry.npm.taobao.org

安装完后验证
cnpm -v

安装hexo

接着用cnpm安装

cnpm install -g hexo-cli

安装完后验证
hexo -v

博客搭建

新建一个文件夹/blog
cd 切换到blog文件夹内

初始化一个博客

hexo init
成功后提示 INFO Start blogging with Hexo!
此时/blog文件夹内生成了一堆博客框架初始文件

启动博客

hexo s
在本地4000端口可以预览

新建博客

hexo n “博客名”
其实就是新建了一个md文件，内容为空

可以编辑里面的内容 以便验证是否修改成功
再次启动可以看到新键的博客

部署博客

登录GitHub

新建一个仓库

new repository
注意：仓库名必须为 “GitHub账号.github.io“

安装hexo的git部署插件

在blog位置
cnpm install –save hexo-deployer-git

修改_config.yml文件

修改内容如下：
”# Deployment
“## Docs: https://hexo.io/docs/deployment.html
deploy:
type: git
repo: git@github.com:jackk1024/jackk1024.github.io.git //仓库地址
branch: master

部署到Git

hexo d

成功后即可访问 jackk1024.github.io
博客部署完成！！！

博客主题切换

找一个喜欢的主题

主题地址：hexo-themes
复制主题的GitHub地址

git clone https://github.com/bulandent/hexo-theme-bubuzou.git themes/bubuzou
//注意加”.git” 可以直接指定克隆到对应文件
克隆完成后修改_config.yml文件:
“# Extensions
“## Plugins: https://hexo.io/plugins/
“## Themes: https://hexo.io/themes/
theme: pln

hexo clean
hexo g
hexo s
hexo d

改完主题需要等一会才能生效
